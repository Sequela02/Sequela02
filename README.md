### What I'm Building

🔥 **[Kiln](https://github.com/sequelcore/kiln)** — Open-source AI orchestration engine. Multi-tenant, YAML-configured, zero vendor lock-in. 8 channel adapters, 5 LLM providers, RAG pipeline, safety pipeline, 25-tool MCP gateway. Published as `@kilnai/*` on npm.

🤖 **[Kilvo](https://kilvo.app)** — AI agent SaaS for SMBs. WhatsApp, Instagram, Messenger, Slack, Web, Email. Built on Kiln. Launching Q2 2026.

🎟️ **[Admit](https://github.com/sequelcore/admit)** — Venue operations & ticketing SaaS targeting Mexico's $5B+ live entertainment market. Live with 2 cinema chains — 18,600+ tickets, ~$572K MXN in revenue since Oct 2025.

📈 **[A.R.T.U.](https://github.com/Sequela02/ehrlich)** — Autonomous algorithmic trading system in Python. LightGBM, FastAPI, Binance API. Market regime detection, Kelly Criterion position sizing, walk-forward validation.
